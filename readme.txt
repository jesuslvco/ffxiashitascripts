# FFXI Ashita Scripts

Este repositorio contiene scripts personalizados para **Ashita V4**, diseñados para facilitar el uso de ítems, el control de un segundo jugador, y la automatización de habilidades y hechizos en Final Fantasy XI.

## Características

- Uso rápido y automatizado de ítems.
- Control y asistencia para un segundo personaje (dual-box).
- Ejecución eficiente de habilidades y magias.
- Scripts personalizables para diferentes situaciones de juego.

## Requisitos

Para utilizar estos scripts es necesario tener instalados los siguientes addons en Ashita V4:

- **UPCAST**
https://github.com/dewiniaid/ffxi-ashita-upcast
- **MULTISEND**
https://github.com/ThornyFFXI/Multisend
- **LUASHITACAST**
https://github.com/ThornyFFXI/LuAshitacast
- **LOTEWHORE**
https://github.com/ThornyFFXI/Lootwhore

Asegúrate de tener estos addons correctamente configurados antes de usar los scripts.

## Instalación

1. Copia los scripts a la carpeta correspondiente de Ashita (`/scripts`).
2. Configura los scripts según tus necesidades en tus macros.

## Uso

Para ejecutar un script, utiliza el siguiente comando en la consola de Ashita:

```
/execute [ruta/del/script] [parámetros]

[ruta] [parametros]
sigue la estructura de los archivos, ejem:
/exec acc/followMe Thunders
#le solicita al personaje secundario Thunders que comience a seguirte
/exec acc/followStop
#le indica a todos los personajes que dejen de seguirte




```

Consulta los comentarios dentro de cada script para instrucciones específicas sobre su funcionamiento y personalización.

---

**Nota:** Estos scripts están pensados para facilitar la experiencia de juego y no deben usarse para infringir las políticas del juego.
